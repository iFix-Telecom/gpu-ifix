## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-13-authz | Elevation of Privilege | hidden-but-callable server action | mitigate | requireOwner() re-checks role==='owner' on EVERY action (D-03); UI hiding cosmetic; Pitfall 3 header forwarding ensures admin gate sees the session |

| T-13-2fa | Tampering | reset-2FA path | mitigate | direct drizzle clear (Pattern 5); never calls /two-factor/enable; CR-01 stays inert (enabled=false post-reset) |
| T-13-audit | Repudiation | admin action logging | mitigate | writeAuditLog one row per admin op (D-09); self-service change-password NOT logged; no secrets in metadata |
