## Engineering Insights Surfaced by Live UAT

1. **`BuildDirector` silently dropped `upstream.Path`** (PR #3) — long-standing bug masquerading as Phase 06.9 model-rewrite gap. Real fix was independent + necessary.
2. **`force-open` CLI was cosmetic for routing** (PR #4) — Plan 04 wired the override into `Set.Execute` only; the dispatcher's `cb.State()` read site bypassed the override entirely. Required new `EffectiveState()` method + 4 call-site updates.
3. **PR #3's path-join broke the openrouter-director path-equality gate** (PR #5) — switched to `HasSuffix` to handle both empty-path and prefixed upstreams.
4. **`docker-compose.yml` defaults can override `.env`** — original compose had `UPSTREAM_LLM_OPENROUTER_MODEL: ${VAR:-qwen/qwen3.5-27b}` + `PROVIDER_ORDER: ${VAR:-novita}` defaults, which silently injected qwen + Novita pin even after operator removed the keys from `.env`. Operator patched compose to empty defaults during UAT.
5. **CI runner timing affects SSE stream test** — `TestChatProxy_SSEStreamingFlushesPerChunk` flaked once on PR #3 build under `-race`; passed on retry. Local 3/3 always pass. Likely CI-machine-specific.
