2. **Broken CDI widespread on 2×3090 hosts** — 55942 + 45778 failed CDI; only 43803 (spike) known-good. With #1 blocking allowlist targeting, broaden lands on broken-CDI or far/slow hosts (California ~50min cold-start). Blocklist catalog seeded: 55942, 45778.

Because of #1+#2, **STT-200 on the 2×3090 topology was not re-confirmed via the gateway** (it WAS proven on the same `:develop` image + machine 43803 via manual create in Plan 02).
