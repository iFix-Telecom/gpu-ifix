| operator-supplied config values → DB | Values arrive later via the gateway write endpoint; this plan defines the bound columns that gate them. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-01 | Tampering | pod_config seed overwriting operator edits on reboot | mitigate | SeedPodConfig uses ON CONFLICT (id) DO NOTHING — idempotent; second boot never overwrites (Task 1 + Task 2 assertion 3) |
| T-17-02 | Denial of Service | spurious NOTIFY storm on idempotent re-save | mitigate | UPDATE trigger gated on `IS DISTINCT FROM OLD` per column (PATTERNS.md line 167) — identical-value save fires no reload |
| T-17-03 | Tampering | infra/secret leakage into pod_config | mitigate | Only the 16 policy knobs are columns; VAST_AI_API_KEY/MINIO_*/DSN are NOT columns (RESEARCH "Infra/secrets OUT OF SCOPE") |
| T-17-SC | Tampering | npm/pip/cargo installs | accept | No package installs in this plan (sqlc + goose already vendored; RESEARCH §Package Legitimacy Audit: N/A) |
