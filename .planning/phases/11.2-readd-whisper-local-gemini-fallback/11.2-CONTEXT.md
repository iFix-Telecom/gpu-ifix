## Critical considerations

1. **Gemini API rate limits:** Free tier 15 RPM / 1500 RPD; paid tier higher but check Google AI Studio quotas. If concurrent requests exceed quota → breaker should open + chain to OpenAI safety net. Phase 11.2 must validate behavior under burst load (smoke test).
2. **Audio format support:** Gemini accepts `audio/wav`, `audio/mp3`, `audio/ogg`, `audio/flac`, `audio/aiff`, `audio/aac`. OpenAI Whisper accepts mostly the same. Director should pass `mime_type` from multipart header verbatim; reject unsupported with HTTP 415.
3. **Audio duration limit:** Gemini 2.5 Flash Lite has ~9.5 hours audio limit per request. Whisper-1 has 25MB file limit (~25 min). For Ifix workflow batch volume, both adequate but document.
4. **Auth model:** Google API key (not OAuth). Store in `UPSTREAM_STT_GEMINI_AUTH_BEARER` env (legacy bearer pattern). Director sends as `x-goog-api-key` header in request, NOT `Authorization: Bearer`. Adapter must translate.
5. **Plan 06.7 D-13 re-assert loop:** Loader.Tier0OverrideURL("stt") must re-include "stt" key when local-stt comes back online; primary reconciler must call OverrideTier0("stt", urls.STT) — code reverted by Phase 11.1, Plan 02 must restore.
6. **Test fixtures:** Phase 11.1 dropped 3-role assertions back to 2; Phase 11.2 must reverse: `len(loader.Snapshot()) == 3` with keys {llm, stt, tts}. The 6 → 5 → 6 model_aliases count cascade must also revert.
