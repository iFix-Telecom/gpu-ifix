  <action>
    Create scripts/chaos/openrouter-iptables-drop.sh per 11-PATTERNS.md openrouter-iptables-drop.sh + Pattern E.

    Required structure:
    1. Shebang + set -euo pipefail.
    2. log() function to stderr with ISO-8601 + tag [openrouter-iptables-drop].
    3. Usage: ACTION="${1:?usage: $0 apply|cleanup}"
    4. DOMAIN constant = "openrouter.ai"
    5. apply branch:
       - IPS=$(dig +short "$DOMAIN" | sort -u). If empty log "no IPs resolved for $DOMAIN" exit 1.
       - For each resolved IP: sudo iptables -I OUTPUT 1 -d "$ip" -p tcp --dport 443 -j DROP -m comment --comment "phase11-chaos-openrouter"
       - Then 2 broad CF CIDR DROPs for rotation insurance:
         sudo iptables -I OUTPUT 1 -d 104.18.0.0/15 -p tcp --dport 443 -j DROP -m comment --comment "phase11-chaos-openrouter"
         sudo iptables -I OUTPUT 1 -d 172.64.0.0/13 -p tcp --dport 443 -j DROP -m comment --comment "phase11-chaos-openrouter"
       - Log "applied N+2 DROP rules tagged phase11-chaos-openrouter".
    6. cleanup branch:
       - Loop: while sudo iptables -S OUTPUT | grep -q "phase11-chaos-openrouter"; do RULE=$(sudo iptables -S OUTPUT | grep -m1 "phase11-chaos-openrouter" | sed 's/^-A/-D/'); sudo iptables $RULE; done
       - Verify: sudo iptables -L OUTPUT --line-numbers | grep -c phase11-chaos-openrouter — expect 0; log "all phase11-chaos-openrouter rules removed".
       - Final: sudo iptables -S OUTPUT | grep -c phase11-chaos-openrouter must be 0; if not, log "WARN: residual rules; manual cleanup needed" exit 1.
    7. Final summary block (Pattern E) to stdout: action taken + rule count delta.

    The script is invoked via ssh from ops-claude: ssh n8n-ia-vm bash -s -- apply < scripts/chaos/openrouter-iptables-drop.sh. Alternatively copy script to n8n-ia-vm and exec there. Document both invocations in script docstring.
  </action>
